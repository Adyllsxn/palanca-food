## 📖 SOBRE O PROJETO

> **PalancaFood** é uma landing page desenvolvida para um restaurante especializado em hambúrgueres artesanais localizado em Luanda. O projeto tem como objetivo apresentar o cardápio digital, facilitar reservas e divulgar promoções do estabelecimento de forma atrativa e moderna.

### ✨ Funcionalidades:
```markdown
✅ Design totalmente responsivo (Desktop, Tablet e Smartphone)
✅ Cardápio interativo com filtros por categoria
✅ Sistema de busca de pratos
✅ Galeria de promoções e banners
✅ Depoimentos de clientes
✅ Blog com receitas e novidades
✅ Formulário de reserva de mesa
✅ Botão "Voltar ao topo"
✅ Animações suaves e hover effects
✅ SEO otimizado
```

## **PRÉ-REQUISITOS**

Antes de começar, certifique-se de ter atendido aos seguintes requisitos:

* [Git](https://git-scm.com/downloads "Download Git") deve estar instalado no seu sistema operacional.

### Executar Localmente

Para executar o **Foodie** localmente, execute este comando no seu git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Adyllsxb/palanca-food.git
```

Windows:

```bash
git clone https://github.com/Adyllsxn/palanca-food.git
```
